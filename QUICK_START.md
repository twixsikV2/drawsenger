# ⚡ Быстрый старт: Защиты мессенджера\n\n## 🎯 За 5 минут\n\n### 1. Импортируйте функции\n\n```typescript\nimport { \n  sanitizeText, \n  validateEmail, \n  validatePassword,\n  detectSQLInjection,\n  logSecurityEvent \n} from './lib/security';\n\nimport { \n  bruteForceProtection,\n  xssProtection,\n  csrfProtection \n} from './lib/advanced-security';\n```\n\n### 2. Валидируйте входные данные\n\n```typescript\n// Email\nif (!validateEmail(email)) throw new Error('Неверный email');\n\n// Пароль\nif (!validatePassword(password)) throw new Error('Слабый пароль');\n```\n\n### 3. Проверяйте на атаки\n\n```typescript\n// SQL Injection\nif (detectSQLInjection(userInput)) throw new Error('Атака!');\n\n// XSS\nif (xssProtection.detectXSS(userInput)) throw new Error('XSS!');\n```\n\n### 4. Санитизируйте данные\n\n```typescript\nconst safe = sanitizeText(userInput);\n```\n\n### 5. Логируйте события\n\n```typescript\nlogSecurityEvent('USER_LOGIN', { userId, email }, 'low');\n```\n\n## 📚 Документация\n\n| Документ | Для кого |\n|----------|----------|\n|
